Questions

Things people ask.

Do I need to know Roblox Studio to use Thalyn?

No. You generate and configure your world inside Thalyn. The exported file opens in Studio, but you don't need Studio skills to create the world — only to add your game logic on top of it.

Will the world work inside my existing Roblox game?

Yes. Thalyn exports .rbxm files (Roblox model format) that you can insert directly into an existing place, as well as complete .rbxl files for a full experience. Both formats are native Roblox.

Can I customise the world after exporting?

Absolutely. The exported file is a standard Roblox file — every part, mesh, and property is editable in Studio. Thalyn gives you a strong starting point, not a locked output.

What's Awen?

Awen is Thalyn's generation credit — a Celtic word for poetic inspiration. Each world generation uses Awen. Your purchase includes a starter pack of Awen credits. Roblox exports don't cost any Awen — only the generation does.

Can I sell games built with Thalyn worlds?

Yes. Everything Thalyn generates belongs to you. You can use it commercially in any Roblox experience without additional licensing or fees.

What's ThalynQuest?

ThalynQuest is Thalyn's in-app progression system. Hidden relics exist within your generated worlds — finding them unlocks deeper creation tools, new style options, and more generation power. The more you explore, the more you can build.
